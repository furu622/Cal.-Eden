---

## 🧠 概要

**Cal. Eden** は、暗算力・処理速度・英語リスニングを同時に鍛えることを目的とした  
ブラウザ上で動作するメンタル算数トレーニングアプリです。

2つのトレーニングモードを搭載しています。

- **Math Flow**：英文と数式を見ながら計算する通常モード
- **Math Flash**：数字が高速表示され、記憶と暗算で合計を求めるフラッシュモード

シンプルな構成ながら、集中力・ワーキングメモリ・暗算力を総合的に鍛える設計になっています。

---

## 🚀 使い方

1. `index.html` をブラウザで開くだけで動作します
2. インストール不要・サーバー不要
3. オフラインでも利用可能

---

## 🎮 モード説明

### Math Flow
- 英文読み上げ（音声あり）
- 数式表示のON/OFF切替可能
- 英文表示のON/OFF切替可能
- 四則演算のモード選択
- 難易度選択あり
- 制限時間あり

### Math Flash
- 数字が高速表示
- 表示終了後に合計を入力
- ワーキングメモリ強化向け
- 難易度に応じて桁・計算難易度が変化
